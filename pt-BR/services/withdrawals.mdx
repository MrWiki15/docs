Um saque é a saída de fundos do Kawi para um destino externo.

## Destinos possíveis

Os saques podem ser:

- Para uma conta bancária.
- Para uma wallet cripto.
- Para outro usuário.
- Para um beneficiário final.
- Para um provedor conectado.

## Exemplo de fluxo

```text
Usuário solicita o saque
          ↓
Kawi valida a operação
          ↓
O motor seleciona a rota adequada
          ↓
A saída é executada
          ↓
O usuário recebe a confirmação
```
