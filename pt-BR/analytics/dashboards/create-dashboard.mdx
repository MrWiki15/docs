Os dashboards agrupam gráficos e métricas relacionados em uma única visualização. Este guia mostra como criar um dashboard do zero e organizar seu layout.

## Criar um novo dashboard

1. Acesse **Analytics → Dashboards**.
2. Clique em **New dashboard**.
3. Informe um nome. Use um nome que reflita o público ou a pergunta que o dashboard responde — por exemplo, "Receita por região" ou "Saúde da fila de Suporte".
4. Opcionalmente, adicione uma descrição para ajudar os colegas a entenderem o propósito do dashboard.
5. Clique em **Create**.

## Adicionar gráficos

Com o dashboard aberto, clique em **Add chart** para inserir uma nova visualização. Cada gráfico é configurado de forma independente e pode extrair dados de qualquer fonte conectada.

## Organizar o layout

Os gráficos podem ser redimensionados e reposicionados arrastando. A grade se ajusta a um layout de 12 colunas. Para reorganizar:

1. Clique em **Edit layout** na barra de ferramentas do dashboard.
2. Arraste os gráficos para novas posições.
3. Redimensione arrastando o canto inferior direito de qualquer gráfico.
4. Clique em **Save layout** ao concluir.

<Info>
  As alterações de layout são salvas separadamente da configuração dos gráficos. Você pode reorganizar sem afetar as consultas subjacentes.
</Info>

## Configurações do dashboard

Acesse as configurações no nível do dashboard pelo menu **⋯** no cabeçalho:

- **Rename** — Atualiza o nome e a descrição do dashboard.
- **Duplicate** — Copia o dashboard para um novo, incluindo todos os gráficos e o layout.
- **Auto-refresh** — Define um intervalo de atualização (30 segundos, 1 minuto ou 5 minutos) para dashboards de monitoramento ao vivo.
- **Delete** — Remove permanentemente o dashboard. Os gráficos não são excluídos e continuam disponíveis como componentes independentes.
