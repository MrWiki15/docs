Este guia leva você de um workspace em branco a um dashboard ativo com pelo menos um gráfico extraindo dados de uma fonte real.

## Antes de começar

- Credenciais da fonte de dados que você planeja conectar (hostname do banco de dados, usuário, senha ou chave de API para uma fonte SaaS)
- Função de Admin ou Member no seu workspace

## Etapas

<Steps>
  <Step title="Conecte uma fonte de dados">
    Acesse **Analytics → Data sources → Add source**. Selecione o tipo de fonte na lista, informe os detalhes da conexão e clique em **Test connection**. Assim que o teste for bem-sucedido, salve a fonte.

    Consulte [Conectar uma fonte de dados](/pt-BR/analytics/connect-data) para ver a lista completa de fontes suportadas e instruções de configuração campo a campo.
  </Step>
  <Step title="Crie um dashboard">
    Acesse **Analytics → Dashboards → New dashboard**. Dê um nome que reflita os dados ou a equipe que ele atende (por exemplo, "Receita semanal" ou "KPIs de Suporte").
  </Step>
  <Step title="Adicione um gráfico">
    Dentro do dashboard, clique em **Add chart**. Selecione a fonte de dados, escolha um tipo de gráfico (barra, linha ou número são bons pontos de partida) e configure os campos dos eixos x e y. Clique em **Save**.
  </Step>
  <Step title="Compartilhe seu dashboard">
    Clique em **Share** no cabeçalho do dashboard. Copie o link para compartilhar com colegas, ou gere um código de incorporação para exibir o dashboard em outra ferramenta. Defina um nível de acesso para controlar quem pode visualizar o link compartilhado.
  </Step>
</Steps>

<Note>
  Se sua fonte de dados exigir um IP em allowlist, adicione os endereços IP de saída da plataforma às regras do seu firewall antes de testar a conexão.
</Note>
