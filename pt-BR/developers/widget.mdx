O widget da Kawi permite integrar fluxos de conversão e remessas em uma plataforma externa com poucas linhas de código.

## Exemplo HTML

```html
<script
  src="https://cdn.kawi.life/widget.js"
  data-app-id="YOUR_APP_ID"
  data-theme="light"
  data-height="720">
</script>
```

## O que o widget permite

- Exibir serviços da Kawi dentro de outro site.
- Processar conversões fiat ↔ cripto.
- Criar fluxos de remessas.
- Permitir onboarding sem construir uma interface do zero.

## Casos de uso

- Fintechs.
- Marketplaces.
- Apps de comunidade.
- Plataformas de remessas.
- Negócios com usuários internacionais.
