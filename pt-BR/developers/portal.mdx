O portal para desenvolvedores é o painel onde as empresas podem gerenciar sua integração.

## Inclui

- API keys.
- Logs.
- Webhooks.
- Estatísticas.
- Status das operações.
- Configuração do widget.
- Ambientes sandbox e produção.
