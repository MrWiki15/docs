O Kawi Engine GitHub será o espaço onde a Kawi poderá publicar:

- SDKs.
- Bibliotecas de integração.
- Exemplos de código.
- Documentação técnica.
- Modelos de integração.
- Ferramentas para desenvolvedores.

## Objetivo

Facilitar que outros desenvolvedores construam sobre a Kawi e acelerar a adoção do ecossistema.
