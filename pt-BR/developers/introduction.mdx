A Kawi permite que empresas e desenvolvedores integrem capacidades de pagamentos, conversão e liquidação diretamente em seus produtos.

Com a API da Kawi, uma empresa pode:

- Criar depósitos.
- Consultar saldos.
- Executar conversões.
- Enviar fundos.
- Receber webhooks.
- Integrar um widget incorporável.
- Usar a Kawi como infraestrutura de remessas ou liquidação.

## Base URL

```text
https://api.kawi.life
```

A API utiliza JSON sobre HTTPS e segue convenções REST.
