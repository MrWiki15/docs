Referência dos principais endpoints da API da Kawi.

## Obter saldo

```http
GET /v1/balance
```

**Resposta:**

```json
{
  "available": "1520.00",
  "currency": "BRL",
  "updated_at": "2026-06-23T12:00:00Z"
}
```

## Criar depósito

```http
POST /v1/deposits
```

**Body:**

```json
{
  "amount": 100,
  "currency": "BRL",
  "method": "PIX",
  "customer_id": "cus_123"
}
```

**Resposta:**

```json
{
  "id": "dep_123",
  "status": "pending",
  "amount": 100,
  "currency": "BRL",
  "payment_method": "PIX"
}
```

## Criar conversão

```http
POST /v1/conversions
```

**Body:**

```json
{
  "from_currency": "BRL",
  "to_currency": "USDC",
  "amount": 1000
}
```

**Resposta:**

```json
{
  "id": "conv_123",
  "status": "processing",
  "from_currency": "BRL",
  "to_currency": "USDC",
  "estimated_amount": "185.40"
}
```

## Criar transferência

```http
POST /v1/transfers
```

**Body:**

```json
{
  "amount": 50,
  "currency": "USDC",
  "destination": "wallet_address_or_user_id"
}
```

**Resposta:**

```json
{
  "id": "trf_123",
  "status": "sent",
  "amount": 50,
  "currency": "USDC"
}
```
