Movimentar dinheiro entre países continua sendo lento, caro e complexo.

## Problemas atuais

- Tarifas altas em transferências internacionais.
- Lentidão na liquidação das operações.
- Fragmentação entre sistemas financeiros tradicionais e blockchain.
- Barreiras de acesso para usuários sem experiência bancária ou cripto.

## A solução

O Kawi resolve esses problemas criando uma camada única em que o dinheiro pode:

1. **Entrar** como fiat.
2. **Movimentar-se** usando infraestrutura blockchain.
3. **Sair** novamente como fiat quando necessário.

Essa camada esconde a complexidade de bancos correspondentes, FX, on/off-ramps e liquidação on-chain por trás de uma API e de um produto simples.
