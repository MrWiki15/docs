<AccordionGroup>
  <Accordion title="A Kawi é um banco?">
    Não. A Kawi é uma infraestrutura tecnológica que conecta sistemas financeiros e blockchain para facilitar operações de dinheiro.
  </Accordion>

  <Accordion title="A Kawi é uma exchange?">
    Não. A Kawi não busca ser uma exchange tradicional. A Kawi simplifica conversões e liquidações para usuários e empresas.
  </Accordion>

  <Accordion title="Preciso entender de cripto para usar a Kawi?">
    Não. A experiência foi desenhada para que o usuário possa operar sem entender de blockchain.
  </Accordion>

  <Accordion title="O que significa liquidação?">
    Liquidação é o momento em que uma operação é concluída e o dinheiro ou valor chega ao destino esperado.
  </Accordion>

  <Accordion title="O que significa infraestrutura de liquidação híbrida?">
    Significa que a Kawi combina sistemas fiat e blockchain para tornar possível a movimentação de valor entre os dois mundos.
  </Accordion>

  <Accordion title="Por que a Kawi usa blockchain?">
    Porque a blockchain permite mover valor rapidamente, com baixo custo e de forma programável.
  </Accordion>

  <Accordion title="Por que a Kawi usa Solana?">
    Porque Solana é rápida, barata e escalável.
  </Accordion>

  <Accordion title="O que é o Kawi Engine?">
    É o motor interno que coordena depósitos, conversões, liquidez, transferências e liquidações.
  </Accordion>

  <Accordion title="Empresas podem integrar a Kawi?">
    Sim. Empresas podem usar a API ou o widget da Kawi para integrar remessas, conversões e liquidação.
  </Accordion>

  <Accordion title="O que é Remessas como Serviço?">
    É o modelo no qual a Kawi permite que outras empresas ofereçam serviços de remessas ou pagamentos usando a infraestrutura da Kawi.
  </Accordion>

  <Accordion title="O que é o widget da Kawi?">
    É uma ferramenta incorporável que permite integrar os serviços da Kawi em outras plataformas com poucas linhas de código.
  </Accordion>

  <Accordion title="Qual é a visão da Kawi?">
    Construir uma camada financeira global onde fiat e blockchain funcionem juntos de forma simples, rápida e acessível.
  </Accordion>
</AccordionGroup>

## Encerramento

A Kawi está construindo uma infraestrutura financeira que une fiat e blockchain em uma única camada.

Seu objetivo não é apenas permitir remessas, mas criar uma base tecnológica para que usuários e empresas possam mover valor globalmente com menos atrito, menor custo e maior velocidade.

A visão final é transformar a Kawi em uma **camada financeira global para mercados emergentes**.
