El laboratorio para desarrolladores es el entorno de pruebas de Kawi.

## Permite

- Simular depósitos.
- Probar conversiones.
- Validar webhooks.
- Testear integraciones.
- Revisar errores.
- Ejecutar flujos sin dinero real.

## Objetivo

Que cualquier desarrollador pueda validar una integración antes de pasar a producción.
