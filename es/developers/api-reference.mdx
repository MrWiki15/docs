Referencia de los endpoints principales de la API de Kawi.

## Obtener balance

```http
GET /v1/balance
```

**Respuesta:**

```json
{
  "available": "1520.00",
  "currency": "BRL",
  "updated_at": "2026-06-23T12:00:00Z"
}
```

## Crear depósito

```http
POST /v1/deposits
```

**Body:**

```json
{
  "amount": 100,
  "currency": "BRL",
  "method": "PIX",
  "customer_id": "cus_123"
}
```

**Respuesta:**

```json
{
  "id": "dep_123",
  "status": "pending",
  "amount": 100,
  "currency": "BRL",
  "payment_method": "PIX"
}
```

## Crear conversión

```http
POST /v1/conversions
```

**Body:**

```json
{
  "from_currency": "BRL",
  "to_currency": "USDC",
  "amount": 1000
}
```

**Respuesta:**

```json
{
  "id": "conv_123",
  "status": "processing",
  "from_currency": "BRL",
  "to_currency": "USDC",
  "estimated_amount": "185.40"
}
```

## Crear transferencia

```http
POST /v1/transfers
```

**Body:**

```json
{
  "amount": 50,
  "currency": "USDC",
  "destination": "wallet_address_or_user_id"
}
```

**Respuesta:**

```json
{
  "id": "trf_123",
  "status": "sent",
  "amount": 50,
  "currency": "USDC"
}
```
