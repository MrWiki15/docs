Kawi Engine GitHub será el espacio donde Kawi podrá publicar:

- SDKs.
- Librerías de integración.
- Ejemplos de código.
- Documentación técnica.
- Plantillas de integración.
- Herramientas para desarrolladores.

## Objetivo

Facilitar que otros desarrolladores construyan sobre Kawi y acelerar la adopción del ecosistema.
