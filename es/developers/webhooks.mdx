Los webhooks permiten recibir notificaciones automáticas cuando ocurre un evento.

## Eventos disponibles

```text
deposit.created
deposit.completed
conversion.started
conversion.completed
transfer.sent
transfer.failed
withdrawal.completed
```

## Ejemplo de payload

```json
{
  "event": "conversion.completed",
  "data": {
    "id": "conv_123",
    "from_currency": "BRL",
    "to_currency": "USDC",
    "status": "completed"
  }
}
```

## Buenas prácticas

- Validar la firma del webhook.
- Responder con HTTP 200.
- Procesar eventos de forma idempotente.
- Guardar logs de eventos recibidos.
