El portal para desarrolladores es el panel donde las empresas pueden gestionar su integración.

## Incluye

- API keys.
- Logs.
- Webhooks.
- Estadísticas.
- Estado de operaciones.
- Configuración del widget.
- Ambientes sandbox y producción.
