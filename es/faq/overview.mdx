<AccordionGroup>
  <Accordion title="¿Kawi es un banco?">
    No. Kawi es una infraestructura tecnológica que conecta sistemas financieros y blockchain para facilitar operaciones de dinero.
  </Accordion>

  <Accordion title="¿Kawi es un exchange?">
    No. Kawi no busca ser un exchange tradicional. Kawi simplifica conversiones y liquidaciones para usuarios y empresas.
  </Accordion>

  <Accordion title="¿Necesito saber de cripto para usar Kawi?">
    No. La experiencia está diseñada para que el usuario pueda operar sin entender blockchain.
  </Accordion>

  <Accordion title="¿Qué significa liquidación?">
    Liquidación es el momento en que una operación se completa y el dinero o valor llega al destino esperado.
  </Accordion>

  <Accordion title="¿Qué significa infraestructura de liquidación híbrida?">
    Significa que Kawi combina sistemas fiat y blockchain para hacer posible el movimiento de valor entre ambos mundos.
  </Accordion>

  <Accordion title="¿Por qué Kawi usa blockchain?">
    Porque blockchain permite mover valor rápido, con bajo costo y de forma programable.
  </Accordion>

  <Accordion title="¿Por qué Kawi usa Solana?">
    Porque Solana es rápida, barata y escalable.
  </Accordion>

  <Accordion title="¿Qué es Kawi Engine?">
    Es el motor interno que coordina depósitos, conversiones, liquidez, transferencias y liquidaciones.
  </Accordion>

  <Accordion title="¿Empresas pueden integrar Kawi?">
    Sí. Empresas pueden usar la API o el widget de Kawi para integrar remesas, conversiones y liquidación.
  </Accordion>

  <Accordion title="¿Qué es Remesas como Servicio?">
    Es el modelo donde Kawi permite que otras empresas ofrezcan servicios de remesas o pagos usando la infraestructura de Kawi.
  </Accordion>

  <Accordion title="¿Qué es el widget de Kawi?">
    Es una herramienta embebible que permite integrar los servicios de Kawi en otras plataformas con pocas líneas de código.
  </Accordion>

  <Accordion title="¿Cuál es la visión de Kawi?">
    Construir una capa financiera global donde fiat y blockchain funcionen juntos de forma simple, rápida y accesible.
  </Accordion>
</AccordionGroup>

## Cierre

Kawi está construyendo una infraestructura financiera que une fiat y blockchain en una sola capa.

Su objetivo no es solo permitir remesas, sino crear una base tecnológica para que usuarios y empresas puedan mover valor globalmente con menos fricción, menor costo y mayor velocidad.

La visión final es convertir a Kawi en una **capa financiera global para mercados emergentes**.
