Kawi funciona conectando dos mundos:

- **Fiat:** dinero tradicional, como reales, dólares u otras monedas locales.
- **Blockchain:** redes y activos digitales como Solana, USDC o USDT.

## La idea central

```text
Fiat para entrar
Blockchain para liquidar
Fiat o cripto para salir
```

Kawi actúa como una **capa de orquestación**. Esto significa que coordina pagos, conversiones, liquidez, rutas y confirmaciones para que el usuario no tenga que hacerlo manualmente.
